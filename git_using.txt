유저네임 이메일설정
git config --global (user.name or user.email)

설정값 확인
git config --list 

추적시작
git init

안에있는파일무조건 지우기
rm -rf .git

브랜치명 수정
git config --global init.defaultBranch main

help문
git help <동사>
git <동사> --help

요약help문
git <동사> -h

branch 이름 변경
git branch -M main

상태확인
git status

스테이지 시키는법
git add 파일이름 
or 
git add .

언스테이징 시키는법
git rm --cached 파일이름

버전만들기
git commit -m "메시지"
git commit -am "메시지

버전들확인
git log --graph

통합 url
토큰+@+주소

git-github연동
git remote add origin 원격주소(토큰값포함)

연동확인
git remote -v

github에 넣기
git push -u origin main
git push origin

원격주소 수정
git remote set-url origin 주소

git에서 특정버전 파일 내용확인
git show 버전 : 파일경로
버전-커밋해시,브랜치,태그

git 내용 비교
git diff
git diff --stage
git diff <commit-hash>
